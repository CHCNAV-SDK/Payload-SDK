# CHCNAV 错误码
本文档主要描述了CHCNAV PSDK中错误码相关的宏定义，以及错误码说明，错误码头文件为chcnav_error.h。

## 宏定义
- 公共错误码
  * 函数返回成功
  ```
  #define CHCNAV_RETURN_OK                       0
  ```
  * 函数返回失败
  ```
  #define CHCNAV_RETURN_ERR_FAILURE              -1
  ```
  * 申请内存失败
  ```
  #define CHCNAV_RETURN_ERR_ALLOC                -2
  ```
  * 执行超时
  ```
  #define CHCNAV_RETURN_ERR_TIMEOUT              -3
  ```
  * 对象不存在
  ```
  #define CHCNAV_RETURN_ERR_NOT_FOUND            -4
  ```
  * 超出指定范围
  ```
  #define CHCNAV_RETURN_ERR_OUT_OF_RANGE         -5
  ```
  * 参数错误
  ```
  #define CHCNAV_RETURN_ERR_PARAM                -6
  ```
  * 系统错误
  ```
  #define CHCNAV_RETURN_ERR_SYSTEM               -7
  ```
  * 系统忙
  ```
  #define CHCNAV_RETURN_ERR_BUSY                 -8
  ```
  * 操作不支持
  ```
  #define CHCNAV_RETURN_ERR_UNSUPPORT            -9
  ```
  * 字符串为空
  ```
  #define CHCNAV_RETURN_ERR_STRING_EMPTY         -10
  ```
  * 字符串不匹配
  ```
  #define CHCNAV_RETURN_ERR_STRING_NOT_MATCH     -11
  ```
  * 未知错误
  ```
  #define CHCNAV_RETURN_ERR_UNKNOWN              -12
  ```
